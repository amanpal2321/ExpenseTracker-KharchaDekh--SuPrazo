# Expense Tracker (KharchaDekh)

## Project Overview
This is a responsive Expense Tracker web application developed using HTML, CSS, and JavaScript. Users can add and manage income and expenses, store data using local storage, and track transactions efficiently.

## Features
- Add income transactions
- Add expense transactions
- Delete transactions
- Data stored using Local Storage
- Responsive Design
- Dark Mode support
- Filter transactions
- Mobile friendly UI

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Local Storage API

## Challenges Faced
1. Implementing local storage
2. Creating responsive UI
3. Handling transaction updates
4. Implementing dark mode
